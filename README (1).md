### 📊 [Data Mining Practicals](https://github.com/fleur12/data-mining)
> *7 hands-on practicals covering the full data mining pipeline — from raw data to patterns*

A structured collection of MSc practicals implemented in Python, each in its own folder with a dedicated README. Covers everything from aggregation and PCA to association rule mining with the Apriori algorithm.

| # | Practical | Key Concepts |
|---|-----------|-------------|
| 1 | Data Aggregation | groupby, sum, mean, resample |
| 2 | Data Integration & Transformation | merge, join, smoothing |
| 3 | PCA | eigenvalues, explained variance |
| 4 | Feature Transformation | log, sqrt, reciprocal transforms |
| 5 | Feature Selection | filter, wrapper, embedded methods |
| 6 | Class Imbalance | oversampling, RandomOverSampler |
| 7 | Association Rule Mining | Apriori, support, confidence, lift |

**Stack:** `Python` `Pandas` `NumPy` `scikit-learn` `mlxtend` `imbalanced-learn` `Seaborn`

[![Repo](https://img.shields.io/badge/View%20Repo-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/fleur12/data-mining)
